# GST Invoice Generator + Profit Dashboard (Frontend Focused)

A frontend-focused web application for generating GST invoices and tracking profit,
designed for small businesses and inspired by tools like myBillBook.

## Features
- GST invoice generation with tax calculation
- Profit and revenue dashboard UI
- Dynamic invoice preview
- Responsive design for mobile and desktop
- Optimized frontend logic for faster updates

## Tech Stack
- HTML
- CSS
- JavaScript

## Growth Perspective
- Built to improve conversion for small business users
- Scope for event tracking and A/B testing
- Can be extended with analytics and automation

## Use Case
Helps SMEs quickly generate invoices and understand daily profit and tax details.

## Future Enhancements
- WhatsApp invoice sharing
- Event-based tracking for paid ads
- Backend & database integration
